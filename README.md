# QR Code Based Library System

## Features
- QR generation for each book copy
- Issue/Return via scanner
- Admin dashboard (create/delete books, view transactions)
- Role-based access (admin/user)

## Tech Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- DB: MongoDB

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
